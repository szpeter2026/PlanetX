## 二、竞品分析：求职精灵

### 2.1 基本信息

| 维度 | 详情 |
|------|------|
| **产品全称** | 求职精灵（Job Seeking Genie / 求职精灵） |
| **所属公司** | 推测为独立创业团队或某招聘平台孵化项目 |
| **产品形态** | 微信小程序 + H5 |
| **核心定位** | AI 驱动的求职助手：简历优化、职位推荐、模拟面试 |
| **目标用户** | 应届生 + 工作 0-5 年初入职场人群 |
| **商业模式** | 基础免费 + 会员订阅（¥19.9-¥49.9/月） |
| **上线状态** | 已上线运营 |

> **备注：** "求职精灵"为通用名称，市场上可能存在多个同名或类似产品。以下分析基于该品类代表性产品的共性特征。如有具体产品需深度调研，可通过测试账号实际体验补充。

### 2.2 产品功能拆解

| 模块 | 功能点 | PlanetX 对标状态 |
|------|--------|-----------------|
| **简历优化** | AI 简历评分、关键词补充、格式美化、多版本生成 | ⚠️ 已有简历解析（`/v1/resume/parse`），缺少主动优化建议 |
| **职位匹配** | 基于简历自动推荐职位、匹配度打分 | ✅ 已有 `/v1/jobs/match` |
| **模拟面试** | AI 面试官提问、语音/文字回答、评分报告 | ❌ 缺失（PlanetX-T空间 规划中 P0） |
| **求职课堂** | 面经、笔试题库、行业科普 | ❌ 缺失（可规划 P2） |
| **投递追踪** | 投递记录、面试日程、offer 管理 | ❌ 缺失（PlanetX 规划中 P1） |
| **个性化推荐** | 基于用户行为+简历的智能推荐算法 | ⚠️ 部分已有（匹配基于文本相似度，缺少行为数据） |

### 2.3 优势分析

| 优势 | 说明 |
|------|------|
| **垂直聚焦** | 100% 聚焦求职场景，功能深度优于通用平台 |
| **上手简单** | 小程序形态，即开即用，无需下载 |
| **AI 成熟度** | 简历优化+面试模拟有成熟的 NLP 方案 |
| **用户认知** | "求职助手"概念已被市场教育 |

### 2.4 劣势与短板

| 劣势 | PlanetX 可攻击点 |
|------|-----------------|
| **单点工具，无生态** | 求职结束后用户必然流失。PlanetX 有情绪陪伴+组队+成长路径，可延长 LTV |
| **无企业端** | 只有 C 端，无法形成供需闭环。PlanetX-T空间 有 B 端 T空间 |
| **品牌调性弱** | 工具属性强，缺乏情感连接。PlanetX 的"星球""重装登陆"有强 IP 潜力 |
| **无社交裂变** | 纯工具，用户不会自发传播。PlanetX 的"3人成团解锁星图"有天然裂变机制 |
| **无心理测评** | 缺少人格匹配维度。PlanetX 的星际人格测试可提供"性格→职业"匹配 |

### 2.5 与 PlanetX 的差异化策略

```
求职精灵：AI 帮你找到工作  →  工具属性，用完即走
    vs
PlanetX：AI 陪你成为更好的自己  →  陪伴属性，长期粘性

核心差异化打法：
1. 求职精灵做"简历→职位"单向匹配
   PlanetX 做"人格+技能+情绪"三维匹配（评测→匹配→陪伴→成长）
2. 求职精灵无社交
   PlanetX 有"3人成团"组队建舱社交裂变
3. 求职精灵求职结束=用户流失
   PlanetX 求职结束=进入成长路径+情绪陪伴，用户长期留存
```

## 三、竞品分析：全民简历

### 3.1 基本信息

| 维度 | 详情 |
|------|------|
| **产品全称** | 全民简历（QuanMin Resume） |
| **所属公司** | 全民简历科技 / 独立运营 |
| **产品形态** | Web 端 + 微信小程序 |
| **核心定位** | 在线简历制作工具：海量模板 + 智能填写 + 一键导出 |
| **目标用户** | 全年龄段求职者（偏传统行业、蓝领、服务业） |
| **商业模式** | Freemium：免费模板有限，高级模板/下载付费（¥9.9-¥29.9） |
| **上线状态** | 已上线多年，成熟运营 |

> **备注：** 同类产品还包括：**简历超人**、**五百丁简历**、**Wondercv 超级简历**、**Canva 简历模板**。

### 3.2 产品功能拆解

| 模块 | 功能点 | PlanetX 对标状态 |
|------|--------|-----------------|
| **简历模板** | 500+ 行业模板、多色系、多风格 | ❌ 无（PlanetX 定位不在此，但需导入简历） |
| **智能填写** | 引导式填写、AI 内容补全、工作经历润色 | ⚠️ 已有文件解析，无智能填写 |
| **简历评分** | AI 综合评分+分维度点评 | ❌ 缺失 |
| **多格式导出** | PDF/Word/HTML/图片 一键导出 | ❌ 无（但 looma-zervi 有文档转换能力可扩展） |
| **简历翻译** | 中英/中日/中韩 简历翻译 | ❌ 无 |
| **投递管理** | 记录投递公司、职位、进度 | ❌ 缺失（PlanetX 规划中 P1） |
| **简历对比** | 多版本简历对比优化 | ❌ 无 |

### 3.3 优势分析

| 优势 | 说明 |
|------|------|
| **模板丰富** | 500+ 模板覆盖几乎所有行业和风格 |
| **简单易用** | 填表式操作，学习成本极低 |
| **用户基数大** | "全民"定位覆盖面广，知名度高 |
| **SEO 成熟** | Web 端搜索流量大 |

### 3.4 劣势与短板

| 劣势 | PlanetX 可攻击点 |
|------|-----------------|
| **纯工具，无智能匹配** | 做完简历后没有下一步。PlanetX 做完简历直接进入匹配+投递 |
| **无 AI 深度** | 模板填写为主，AI 能力弱。PlanetX 的 looma-zervi 后端有 LiteLLM+pgvector 全栈 AI |
| **用户价值结束时点早** | 简历做完即流失。PlanetX 有完整的求职→面试→入职→成长链路 |
| **无 B 端连接** | 简历做出来不知道投给谁。PlanetX-T空间 直接对接企业 JD |
| **品牌老化** | "全民简历"调性偏传统，对 00 后/GEN-Z 吸引力弱 |

### 3.5 简历工具赛道全景对比

| 产品 | 定位 | 价格 | 核心优势 | 核心短板 |
|------|------|------|----------|----------|
| **全民简历** | 通用模板工具 | ¥9.9-29.9 | 模板多、上手简单 | AI 弱、无匹配 |
| **超级简历 Wondercv** | 精英极简风 | ¥19.9-49.9/月 | 设计感强、有社区 | 仅互联网行业，覆盖面窄 |
| **五百丁** | 中高端模板 | ¥29.9-99.9 | 模板质量高 | 贵、AI 弱 |
| **Canva 简历** | 设计工具附属 | ¥0-99.9/月 | 设计自由度极高 | 非求职专用，无匹配 |
| **PlanetX（规划）** | **AI 全链路** | ¥0-29.9/月 | 解析→匹配→投递→面试→成长 | 模板少（可补齐） |

## 四、竞品分析：MBTI 心理测评赛道

### 4.1 赛道概述

MBTI（Myers-Briggs Type Indicator）在中国 GEN-Z 群体中已成为现象级社交货币。从"你的 MBTI 是什么"成为年轻人破冰话题，到各平台 MBTI 相关内容播放量数百亿次，MBTI 已经从心理学工具进化为**社交身份标签**和**消费决策依据**。

| 指标 | 数据 |
|------|------|
| 微博 #MBTI# 话题阅读量 | 50 亿+ |
| 小红书 MBTI 相关笔记 | 1000 万+ |
| 抖音 MBTI 相关视频播放 | 300 亿+ |
| B站 MBTI 二创视频 | 日均新增 500+ |

### 4.2 主流 MBTI 产品竞品分析

### 4.3 MBTI 赛道的核心机会

```
痛点1：测评完了就完了，没有后续
    → 所有 MBTI 产品都是"测完分享→结束"的单次体验
    → PlanetX 将 MBTI 作为入口，后续连接职位匹配+情绪陪伴+成长路径

痛点2：MBTI 的职业推荐是静态的"人格→岗位"映射表
    → 没有真实 JD、没有 AI 多维打分、没有企业端
    → PlanetX 做"人格+技能+简历"三维匹配到真实企业 JD

痛点3：MBTI 没有社交裂变产品化
    → 虽然 MBTI 天然有社交传播属性，但没有产品做"组队""解锁"
    → PlanetX 的"3人成团解锁隐藏星图" = MBTI 人格组队探险

痛点4：MBTI 产品没有 B 端变现
    → 全部靠 C 端付费，ARPU 低
    → PlanetX-T空间 有 B 端企业付费（招聘SaaS），C端可免费
```

## 五、竞品分析：AI 招聘 SaaS 赛道

### 5.1 赛道概述

T空间 B端产品的直接竞品来自企业招聘 SaaS 市场。该市场在中国已较为成熟，主要玩家包括 Moka、北森、大易、薪人薪事等。

### 5.2 主要竞品

| 产品 | 成立 | 融资 | 核心定位 | 目标客户 | 2025 ARR 估算 |
|------|------|------|----------|----------|-------------|
| **Moka** | 2015 | D轮 $1亿+ | 智能化招聘管理系统 | 互联网/科技企业 | ¥10亿+ |
| **北森 (Beisen)** | 2002 | 港股上市 | 一体化 HR SaaS | 中大型企业 | ¥8亿+ |

### 5.4 T空间 的差异化突破点

```
现有招聘 SaaS 的问题：

2. 没有 AI 模拟面试官（这是 T空间海报的四大核心能力之一）

5. 价格昂贵——Moka/北森年费 ¥2万-¥50万，中小企负担重

T空间 的差异化打法：
1. "前30家永久免费"——以中小企业为切入点，大厂看不上
2. AI 模拟面试 + 知识库 RAG —— 两个 Moka/北森 没有的 AI 杀手功能
3. C端 PlanetX 自带求职者流量 —— 企业不用额外买招聘广告
4. 心理测评 → 职业匹配 —— 人格维度的匹配是传统招聘系统做不到的
5. ×× 的定价模型 —— 中小企的可负担 SaaS
```
